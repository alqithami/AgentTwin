# Supplementary material for AgentTwin

This directory contains auxiliary evaluation tables associated with the Journal of Process Control article:

**AgentTwin: a multi-agent digital-twin testbed for supervisory operating-mode scheduling and residual regulatory control on the Tennessee Eastman Process with a solver-backed safety shield**

The tables support the manuscript's Data and code availability statement and include:

1. matched-reset reevaluation tables,
2. PI shield on/off sensitivity tables,
3. evaluation-only terminal-penalty and post-shutdown-fill sensitivity tables.

These supplementary tables are provided as publication-supporting artifacts. They do not change the main reported results in the article.
